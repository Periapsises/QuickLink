## Installation

You can easily install this package from NuGet using the following methods:

# [NuGet Package Manager](#tab/nugetpackagemanager)

```bash
Install-Package QuickLink
```

# [.NET CLI](#tab/dotnetcli)

```bash
dotnet add package QuickLink
```
